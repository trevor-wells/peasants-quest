# **peasant's quest**

a CLI text adventure inspired by homestar runner's 2004 graphical text adventure under the same name

## **installation**

to set up the app on your local machine, navigate to the project folder in your terminal and type:

```
pipenv install
pipenv shell
cd lib/db
python models.py
python seed.py
python cli.py
```
